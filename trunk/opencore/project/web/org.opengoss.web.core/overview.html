WebCore plugin provides a framework and implementation to deploy Browser/Server structure application on OpenCore and OSGi.Clients could use this framework to create pure plugin-based web applications.

In this framework, a web applications may contain one or more web plugins. Each Web plugin is an OpenCore plugin(or OSGi bundle) that provides static web pages and servlets, defines its own "WEB-INF/web.xml" file, implements a plugin activator that extends {@link org.opengoss.web.core.WebPluginActivator}
