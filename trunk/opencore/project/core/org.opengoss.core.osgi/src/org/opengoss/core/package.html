Core plugin of OpenCore SOA Platform to support Ioc and ExtensionPoint based on OSGi.

Core Plugin Offers:
  • Plugin level IoC: property injection, constructor injection
  • ExtensionPoint and Extension
  • Parse the "META-INF/plugin.xml", inject dependecies between services and bind the extentionPoint and extentions.

Plugins built on OpenCore SOA platform should:
  • Configure a "META-INF/plugin.xml" file
  • Code an activator class that extends {@link org.opengoss.core.PluginActivator}.
